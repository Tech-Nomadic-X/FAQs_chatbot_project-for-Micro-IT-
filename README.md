# 🎯 Internship FAQ Chatbot – Micro IT Services

This is an **AI-powered FAQ chatbot** built using **Dialogflow**, **HTML/CSS/JS**, and a **Flask** backend. It helps answer internship-related queries from students at **Micro IT Services** in real-time.

---

## 🚀 Features

- 🔍 Predefined FAQ automation using Dialogflow Intents
- 🌐 Web UI for chat embedded via HTML/CSS/JS
- 🧠 AI-driven responses using Dialogflow’s NLP engine
- 🔄 Flask backend for integration
- 📱 Mobile-responsive UI
- 🖼️ Clean, minimal interface with custom chatbot icon

---

## 🧰 Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Python (Flask)
- **AI/NLP**: Dialogflow ES
- **Others**: Git, GitHub

---

## 🛠️ Setup Instructions

1. **Clone the Repo**

```bash
git clone https://github.com/Tech-Nomadic-X/FAQs_chatbot_project-for-Micro-IT-.git
cd FAQs_chatbot_project-for-Micro-IT-

2. Install Dependencies

pip install -r requirements.txt

3. Run the Flask App

python app.py

4. Open in Browser

Visit http://127.0.0.1:5000 to interact with the chatbot.
